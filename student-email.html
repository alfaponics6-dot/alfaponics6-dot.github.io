Estimado/a colega,

Me comunico contigo como compañero estudiante de la Universidad EARTH para compartirte un servicio profesional que creo puede aportarte un valor real — especialmente al acercarte a tu último año y comenzar a construir tu identidad profesional.

Diseño y desarrollo soluciones digitales de nivel empresarial — desde sitios web de portafolio profesional y aplicaciones web hasta software para Android, iOS y escritorio — adaptadas a las necesidades y presupuesto de los estudiantes universitarios.

Servicios Ofrecidos

Sitio Web de Portafolio con Dominio Propio

Un sitio web de marca personal completamente personalizado, publicado en tu propio dominio (ej. tunombre.com). Diseñado para establecer una presencia profesional de alto nivel para estudiantes de 4.º año antes de graduarse, aplicar a empleos y acceder a oportunidades de investigación.

Aplicaciones Móviles Android e iOS

Aplicaciones móviles nativas y multiplataforma listas para publicación real. Ideales para herramientas de investigación académica, recolección de datos en campo, gestión agrícola o cualquier concepto que quieras hacer realidad.

Software de Escritorio — Windows y macOS

Aplicaciones de escritorio independientes para análisis de datos, gestión de laboratorio, procesamiento de investigaciones o automatización — diseñadas para funcionar sin conexión y manejar operaciones complejas sin navegador.

Aplicaciones Web y Paneles de Control

Aplicaciones web full-stack — plataformas de gestión, paneles interactivos, sistemas de inventario y herramientas de datos — accesibles desde cualquier dispositivo con navegador, sin necesidad de instalación.

Proyectos Activos y Experiencia Comprobada

Asistente Digital Forestal — Universidad EARTH En progreso

Aplicación móvil en desarrollo activo para la Universidad EARTH — con soporte para gestión forestal, recolección de datos en campo e identificación de especies.

Plataforma de Inventario y Préstamo de Materiales — IGA En progreso

Sistema completo de gestión de inventario y préstamo de equipos en tiempo real para el departamento de IGA — seguimiento de activos, registro de préstamos a estudiantes y generación de informes de uso.

Software de Análisis Genético Completado

Software de escritorio especializado desarrollado para el procesamiento y análisis de datos genéticos — demostrando dominio en computación científica e ingeniería de software orientada a dominio.

Sitios Web de Portafolio para Estudiantes de 4.º Año En curso

Sitios web de marca personal diseñados a medida con dominio registrado — ayudando a estudiantes de último año a presentarse como candidatos de primer nivel antes de graduarse.
